Günstiges Hosting von statischen Inhalten mit CloudFront und AWS S3

Ein Projekt auf GitHub

Als Basis für die Inhalte der Website nutze ich GitHub. Besagtes Projekt ist "public", also mit öffentlichen Quellen angelegt - weder der statische Code noch das Deployment offenbart Geheimnisse!
Zugangsdaten wie AWS Tokens sind zwar im Projekt hinterlegt, jedoch nicht für Außenstehende einsehbar.

S3 Bucket + CloudFormation

Sowohl der Bucket auf S3 als auch die "CloudFront Distribution"

Hosted Zone auf Route53

Deployment der Inhalte auf AWS S3

name: Deploy Website to AWS S3

on:
  push:
    branches:
    - master

jobs:
  deploy:
    runs-on: ubuntu-latest
    steps:
    - uses: actions/checkout@master
    - uses: jakejarvis/s3-sync-action@master
      with:
        args: --acl public-read --follow-symlinks --delete
      env:
        AWS_S3_BUCKET: ${{ secrets.AWS_S3_BUCKET }}
        AWS_ACCESS_KEY_ID: ${{ secrets.AWS_ACCESS_KEY_ID }}
        AWS_SECRET_ACCESS_KEY: ${{ secrets.AWS_SECRET_ACCESS_KEY }}
        AWS_REGION: 'eu-central-1'   # optional: defaults to us-east-1
        SOURCE_DIR: 'website'
